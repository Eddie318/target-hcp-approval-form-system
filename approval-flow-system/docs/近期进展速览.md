## 近期进展速览（PC/H5）

- **链路更新**：新增目标医院链路包含 C&D（BISO2 后）；新增关联药房仅“其他+关联”插入 C&D；医院区域调整链路改为 DSM→RSM→BISO1→RSD→BISO2。
- **附件规则**：取消目标医院/取消关联药房附件改为非必填；新增关联药房“其他+A类”提交必填、“其他+关联”在 C&D 同意前必填。
- **新增关联药房**：先选医院（指标表），展示该医院已关联药店；已关联≥3 家禁止新增；白名单候选会剔除指标表已关联药店。
- **登录切换**：前端改为读取 `tokenInfo`，走 `/auth/saas-user` + `/auth/resolve-user` 映射岗位号/角色/大区。
- **归档查看**：PC 归档增加状态筛选、提交/审批时间筛选、全文搜索、大区列；导出支持 5 个 Sheet。

> 详尽版本见《规则与功能汇总.md》。
